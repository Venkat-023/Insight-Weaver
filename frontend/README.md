# Scientific Discovery Copilot Frontend

React + Vite frontend for the FastAPI backend.

## Run

```powershell
cd E:\Projects\Gemma-hackathon-main\frontend
npm install
npm run dev
```

Open `http://127.0.0.1:5173`.

The UI expects the backend at `http://127.0.0.1:8000/api/v1` by default. You can change it in the sidebar.
